# 🚀 GitFlex

**Деплой любых сайтов за секунды - без сервера!**

GitFlex - это революционное приложение, которое позволяет мгновенно разворачивать и просматривать сайты прямо из репозиториев GitHub, GitLab, Bitbucket и любых других Git-платформ. Всё работает полностью в браузере, без какого-либо сервера!

![GitFlex Banner](https://img.shields.io/badge/GitFlex-Deploy_Anything-blue?style=for-the-badge&logo=git)

## ✨ Особенности

### 🔥 Молниеносный деплой
- Загрузка через raw CDN обеспечивает максимальную скорость
- Мгновенный предпросмотр вашего сайта
- Никаких задержек и ожиданий

### 🛡️ Полная безопасность
- Никакого сервера - всё работает в вашем браузере
- Ваш код никогда не сохраняется на наших серверах
- Приватность гарантирована

### 🌍 Универсальная поддержка
- **GitHub** - полная интеграция с API
- **GitLab** - поддержка всех репозиториев
- **Bitbucket** - работайте с любимой платформой
- **Любые другие Git-платформы** - Gitea, Gogs и другие

### 💻 Любой стек технологий
- HTML/CSS/JS
- React
- Vue.js
- Angular
- Svelte
- И любые другие фронтенд-фреймворки

### 📱 Адаптивный дизайн
- Идеально работает на desktop
- Оптимизировано для мобильных устройств
- Планшеты поддерживаются

## 🎯 Как использовать

### Быстрый старт

1. **Откройте GitFlex**
   - Просто откройте `index.html` в любом современном браузере

2. **Вставьте ссылку на репозиторий**
   ```
   https://github.com/user/repository
   ```

3. **Нажмите "Задеплоить"**
   - Приложение автоматически определит платформу
   - Загрузит файлы через raw
   - Отобразит ваш сайт в предпросмотре

4. **Готово!**
   - Просматривайте сайт
   - Поделитесь ссылкой с друзьями
   - Откройте в новой вкладке

### Примеры URL

```bash
# GitHub
https://github.com/twbs/bootstrap
https://github.com/vuejs/vue

# GitLab
https://gitlab.com/gitlab-org/gitlab

# Bitbucket
https://bitbucket.org/team/repository

# Любая другая платформа
https://your-git-platform.com/user/repo
```

### Как это работает

1. **Парсинг URL** - определяем платформу и репозиторий
2. **API запрос** - получаем информацию о репозитории
3. **Загрузка файлов** - используем raw CDN платформы
4. **Поиск index.html** - находим главный файл
5. **Предпросмотр** - отображаем в iframe

## 🛠️ Разработка

### Требования
- Современный браузер (Chrome, Firefox, Safari, Edge)
- Доступ к интернету (для загрузки репозиториев)

### Запуск

```bash
# Просто откройте файл в браузере
open index.html

# Или используйте локальный сервер
python -m http.server 8000
# Затем откройте http://localhost:8000
```

## 🎨 Кастомизация

### Изменение темы

В `styles.css` измените CSS переменные:

```css
:root {
    --primary-color: #6366f1;      /* Основной цвет */
    --secondary-color: #ec4899;    /* Вторичный цвет */
    --accent-color: #06b6d4;       /* Акцентный цвет */
}
```

### Добавление платформы

В `script.js` добавьте конфигурацию:

```javascript
const platforms = {
    yourPlatform: {
        api: 'https://api.your-platform.com',
        raw: 'https://your-platform.com',
        patterns: [/your-pattern/i]
    }
};
```
