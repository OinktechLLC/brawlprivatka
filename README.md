# ПриваткаБравл (PrivatkaBrawl)

🔫 **Шутер от первого лица в стиле Brawl Stars + Standoff 2 + Venge.io**

Веб-игра на Unity (WebGL) с ботами, прокачкой персонажа, инвентарем и боевым пропуском.

## 🎮 Особенности
- **Геймплей**: Динамичные перестрелки на картах в стиле Brawl Stars.
- **Графика**: Стиль Low Poly / Venge.io (требует замены ассетов).
- **Боты**: Простой ИИ противников.
- **Прогрессия**:
  - Локальное сохранение (localStorage).
  - Сбор "Куки" (монет) во время боя.
  - Боевой пропуск (Battle Pass).
  - Инвентарь скинов и оружия.
- **Меню**: Главное меню в стиле Brawl Stars.

## 🛠 Технологии
- **Движок**: Unity 2022+
- **Язык**: C#
- **Платформа**: WebGL (браузер)

## 🚀 Как запустить
1. Скачайте и установите [Unity Hub](https://unity.com/download).
2. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ВАШ_НИК/PrivatkaBrawl.git
   ```
3. Откройте папку проекта в Unity Hub.
4. Нажмите **Play** в редакторе или сделайте билд: `File -> Build Settings -> WebGL -> Build`.

## 📂 Структура проекта
- `Assets/Scripts/Player` - Логика игрока и стрельбы.
- `Assets/Scripts/Bot` - ИИ ботов.
- `Assets/Scripts/UI` - Интерфейс, меню, инвентарь.
- `Assets/Scripts/System` - Сохранения, Battle Pass.

## 🎨 Графика и Ассеты
В текущей версии используются примитивы (кубы/капсулы). Для полной графики как в Venge.io:
1. Скачайте модели оружия и персонажей (Mixamo, Unity Asset Store).
2. Замените объекты в префабах `Player` и `Bot`.
3. Настройте материалы и освещение.
